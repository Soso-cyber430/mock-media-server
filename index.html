Welcome to the Media API

Use the following endpoints to interact with the data:

GET

  • /movies
  • /songs
  • /series

POST

Send JSON body like:

{
  "title": "Interstellar",
  "genre": "Sci-Fi",
  "year": "2014"
}

PUT

Same structure as POST. Updates if title exists.

DELETE

Send JSON body:

{
  "title": "Spirited away"
}
