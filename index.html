API Documentation

This is a simple mock media server. Here are the available endpoints:

GET /movies, /songs, /series

Returns a list of media items of the specified type (movies, songs, or series).

POST /movies, /songs, /series

Add a new item to the specified media type. Example body:

      { 
        "title": "Movie Title",
        "genre": "Action",
        "year": 2023
      }

PUT /movies, /songs, /series

Update an existing item. Example body:

      { 
        "title": "Updated Title",
        "genre": "Drama",
        "year": 2024
      }

DELETE /movies, /songs, /series

Delete a media item by title. Example body:

      { 
        "title": "Movie Title"
      }
